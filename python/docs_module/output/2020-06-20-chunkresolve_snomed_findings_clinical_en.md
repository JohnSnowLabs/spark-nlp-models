## Description
Entity Resolution model Based on KNN using Word Embeddings + Word Movers Distance


{:.h2_title}
## Prediction Domain
Snomed Codes and their normalized definition with `clinical_embeddings`

{:.h2_title}
## Data Source
Trained on SNOMED CT Findings
http://www.snomed.org/
